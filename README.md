# Mine Sweeper
Implementation of the Mine Sweeper game in Java

## Instructions
clone the repository
```
git clone https://github.com/juandiegotor/MineSweeper.git
```
Move into the source directory
```
cd MineSweeper/src
```
Run
```
javac Game.java
java Game
```
> Game developped using openjdk 10.0.2

Done!
